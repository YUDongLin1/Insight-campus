# Insight-campus
校园情报官 Insight Campus，面向大学生的 AI 校园信息处理工作台，把 QQ 群通知、校园海报、DDL、宣讲会和社团信息自动转化为任务、日历、笔记、分享卡片和行动建议。

# 面向腾讯 PCG 校园 AI 产品创意大赛的 AI 产品 Demo。
校园情报官 Insight Campus 是一款面向高校学生的 AI 校园信息工作台，帮助学生把 QQ 群通知、校园海报、课程 DDL、宣讲会、社团活动和比赛信息，自动整理成 任务、洞察、日历、笔记、QQ 频道分享卡片和模拟面试问答。

# 产品定位
大学生日常信息高度分散：QQ 群、公众号、课程平台、学院通知、线下海报、班委转发、社团群和实习群不断刷屏。学生真正缺少的不是又一个提醒工具，而是一个能把碎片通知转成可执行行动的 AI 工作流。

# 目标用户
高校本科生、研究生
经常被课程 DDL、讲座、比赛、社团通知打断的学生
正在准备比赛、求职、实习、社团活动的同学
需要转发通知、组织活动、提醒成员的班委和社团负责人

# 解决的痛点
信息太分散：校园通知散落在 QQ 群、公众号、课程平台、海报和截图中，学生很容易漏掉时间、地点、报名方式和截止日期。
通知不能直接执行：传统通知只是文本或图片，学生仍然要自己提取时间、拆分任务、同步日历和提醒队友。
海报截图难处理：校园活动常以海报形式传播，传统 OCR 只能识别文字，不能理解标题重点、活动类型和视觉层级。
资料难沉淀：比赛、求职和活动准备过程中的灵感、洞察、行动清单和访谈材料，容易丢在聊天记录里。

# 核心功能
1. 情报中心
支持粘贴 QQ 群通知、课程 DDL、活动公告、比赛信息等文本，自动提取：
主题、时间、地点、类型、优先级、下一步行动

2. 海报/截图识别
支持上传校园海报或截图。后端版本可调用智谱 BigModel 多模态模型，识别图片中的文字和视觉信息；静态演示版内置本地模拟能力，方便评委直接体验。

3. 智能洞察
根据通知类型生成准备建议和 PCG 场景联动建议，例如：
QQ/QQ 频道分发
腾讯文档沉淀资料
腾讯新闻补充背景
腾讯视频保存 Demo 或讲座内容
大模型辅助模拟答辩和面试

4. 情报日历
将明确的任务同步到日历，支持：

倒计时
日期查看
手动添加日程
删除日程
中文时间解析
5. 随身笔记
支持把产品洞察、路演材料、访谈结论和个人备忘沉淀为笔记。静态版使用浏览器 localStorage 保存。

6. 搭子雷达
基于共同日程和兴趣标签，展示学习、比赛、求职或社团搭子匹配的产品概念。

7. 模拟面试 Agent
围绕产品定位、用户痛点、AI 能力、PCG 生态和商业化进行追问，帮助准备比赛路演和产品面试。
